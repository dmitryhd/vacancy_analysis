Ведущий UNIX-разработчик

 

Функциональные обязанности

  • Разработка ПО на языках C/C++ под ОС Linux.
  • Доработка различных частей существующего кода в соответствии с требованиями.
  • Исправление ошибок, поддержка существующего кода.
 

Обязательные требования

  • Отличное знание языков С/C++
  • Отличное знание архитектуры систем семейства Linux
  • Хорошее знание Unix API: демоны, межпроцессное взаимодействие (IPC), мультипотоковые программы
  • Базовые знания протоколов семейства TCP/IP, опыт разработки сетевых программ
  • Базовые знания библиотек: STL, boost
  • Владение основными средствами разработки и контроля версий под Linux: bash, make, gcc, ld, gdb, valgrind, cvs и т.д
  • Умение разобраться в чужом коде, умение работать в команде
  • Базовые навыки администрирования Linux-систем
 

Приветствуется:

  • Опыт разработки программно-аппаратных комплексов (терминальные решения и т.п.)
  • Опыт проектирования архитектуры ПО, шаблоны проектирования
  • Знание архитектуры и опыт разработки ПО на других системах семейства Unix
  • Опыт разработки кроссплатформенного ПО под различные аппаратные архитектуры (ARM, MIPS и т.д.) и операционные системы
  • Теоретические знания и практический опыт разработки драйверов под Linux и другие Unix-системы
  • Опыт работы с Security Tokens различных производителей
  • Опыт работы разработчиком в коммерческих компаниях

  • Испытательный срок 3 месяца.
